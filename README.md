## Usage
Clone and install dependencies.
```
git clone https://github.com/pearllaw/optimized-cooking.git
cd optimized-cooking/
npm install
```
Create a '.env' file in the project root. Example:
```
PORT=3000
```
Run the app
```
npm run watch
```
